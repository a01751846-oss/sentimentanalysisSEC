# SEC Investment Sentiment Analyzer

AI-powered sentiment analysis for SEC-style financial documents using Qwen2 0.5B Instruct.

## Features

- Local LLM inference
- Financial sentiment analysis
- Investment recommendation generation
- Risk and opportunity extraction
- Python + Transformers pipeline

## Tech Stack

- Python
- Hugging Face Transformers
- Qwen2 0.5B Instruct
- Requests

## Example Output

```text
Overall Sentiment: Positive

Risks:
- Regulatory uncertainty
- Market volatility

Positive Indicators:
- Strong revenue growth
- Cash reserves

Investment Opinion:
Potentially attractive long-term investment.
```

## Run Locally

```bash
pip install transformers torch accelerate requests
```

```bash
python app.py
```
